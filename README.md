# Monorepo

Combines the following upstream projects via `git subtree`,
preserving their full commit history in subfolders:

- [`LiveExec32`](./LiveExec32) — from https://github.com/LiveContainer/LiveExec32.git (Apache-2.0)
- [`touchHLE`](./touchHLE) — from https://github.com/touchHLE/touchHLE.git (MPL-2.0 source / GPLv3+ binaries)

Each subfolder retains its own LICENSE file. See that file for the terms
governing the code in that subfolder.
